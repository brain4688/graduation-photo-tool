# 📷 毕业合影美化工具

> 纯前端毕业合影照片美化工具，支持自定义底栏色块、文字排版、滤镜效果，一键导出高清 PNG。

🔗 **在线使用**：https://brain4688.github.io/graduation-photo-tool/

---

## ✨ 功能特性

### 核心功能
- 📤 本地图片上传（JPG/PNG），等比例缩放不变形
- 📝 底栏自定义文字，支持多行换行、自动换行适配宽度、兼容繁体
- 🎨 底栏颜色可选：浅红、酒红、深暗红、纯黑、深蓝、深棕
- 📍 底栏位置可选：顶部 / 底部
- 🔤 文字字体可选：宋体古风、黑体、楷体、仿宋、微软雅黑
- 🎨 文字颜色可选：白色、金色、浅金色、香槟金等 10 种

### 滤镜效果
- 原图无滤镜
- 黑白复古
- 暖调复古
- 胶片颗粒
- 青灰电影感

### 导出与适配
- 💾 一键导出高清 PNG（原图分辨率，清晰度不损失）
- 📱 响应式布局，自动适配电脑 / 手机
- 🔄 参数修改实时预览，附带手动刷新按钮

---

## 🚀 使用方式

### 本地使用
1. 下载 `index.html` 文件
2. 用 Chrome 或 Edge 浏览器双击打开
3. 上传照片 → 调整参数 → 导出 PNG

### 线上部署
支持部署到 GitHub Pages、Vercel、Netlify 等静态托管平台。
