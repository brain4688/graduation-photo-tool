一、本地打开方法

1. 将本 index.html 文件保存到电脑任意位置。

2. 双击文件，使用 Chrome 或 Edge 浏览器打开即可使用，无需安装任何软件。

3. 上传本地照片，调整参数，点击"导出高清 PNG"保存成品。

二、线上部署分享步骤

GitHub Pages：将文件上传至 GitHub 仓库 → Settings → Pages → 选择分支 → 生成公网链接。

Vercel：注册 Vercel 账号 → Import 项目 → 自动部署 → 获得 xxx.vercel.app 链接。

DevFile / Netlify：直接拖拽 index.html 文件到平台即可部署。

三、故障自查方案

预览空白：①确认已上传图片 ②检查浏览器是否为最新版 ③尝试点击"手动刷新预览" ④检查图片文件是否损坏。

图片加载失败：确认文件格式为 JPG 或 PNG，文件大小建议不超过 20MB。

导出失败：确认浏览器允许下载权限，部分浏览器需手动允许弹窗。

微信内置浏览器兼容问题：微信内置浏览器对 Canvas API 支持不完善，建议使用系统浏览器（Chrome/Edge）打开。

四、隐私说明

所有图片处理均在您的浏览器本地完成，不会上传至任何服务器。本工具为纯静态页面，无后端服务。
